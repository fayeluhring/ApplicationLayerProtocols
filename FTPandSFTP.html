File Transfer Protocol (FTP)

FTP is one of the earliest and most common forms of transferring files through the internet ("File Transfer Protocol (FTP) in Application Layer", 2024, 1). "FTP is a client server protocol that has two communication channel, command channel, for conversation control and data channel for file content" ("File Transfer Protocol (FTP) in Application Layer", 2024, 6). While FTP is one of the most common forms of transferring files through the internet, it was not designed with security in mind.

Secure File Transfer Protocol (SFTP)

SFTP was then created! It is the secure version of FTP. It encrypts your data before sending the files ("File Transfer Protocol (FTP) in Application Layer", 2024, 18).

  • Two communication channels in FTP
  • FTP does not encrypt data
  • SFTP adds encryption
  • Used widely for secure file sharing
